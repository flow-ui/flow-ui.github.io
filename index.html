Flow-UI 是一套高定制性UI框架，基于Seajs/jQuery构建，内置CSS组件、JS插件、模块化开发体系、自动化开发工具、常见场景解决方案。

当你需要开发拥有独特风格的WEB站点时，Flow-UI是你的不二之选，配合Flow-CLI可以获得极高的UI定制性；同时借助丰富的JS插件，Flow-UI也可以满足中台开发需求，此时你可以获得相比MVVM方案更好的兼容性。

快速开始 下载

 安装

使用编译后文件：

npm i @tower1229/flow-ui

自动化开发：

npm i front-flow-cli -g

 特性

  • - 极高的定制自由度，为定制UI而生
  • - 海纳百川，同时支持模块化/非模块化JS加载
  • - 模块本地存储，Hold住你的请求数
  • - html + css + js = 自定义组件，静态稿也能组件化开发
